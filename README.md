# Warthogs
Warthogs is a street-fighter style game with different planets as the maps where the player can use their hands to hover over the screen to hit bubbles that allow them to attack the enemy.
## Features
* Play as 2 different characters and engage in combat against an enemy.
* Use your hand to control the character's attacks by hovering over the screen.
* Each character has unique abilities and animations.
* Play on 2 different planets
